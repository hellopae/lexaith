# LexAI Thailand 🏛️
### ผู้ช่วยกฎหมายไทย · AI-Powered Thai Legal Assistant

LexAI Thailand เป็นแอปผู้ช่วยกฎหมายไทย ทำงานเป็นไฟล์ HTML ไฟล์เดียว ใช้งานได้ทันทีในเบราว์เซอร์ โดยไม่ต้องติดตั้งอะไรเพิ่มเติม

---

## ✨ ฟีเจอร์หลัก

- **🤖 AI Chat** — ถามปัญหากฎหมายด้วยภาษาพูดธรรมดา ขับเคลื่อนด้วย Google Gemini AI
- **📚 ห้องสมุดกฎหมาย** — กฎหมายไทย 6 หมวดหมู่ อธิบายด้วยภาษาง่าย (แรงงาน, ที่ดิน, ครอบครัว, ผู้บริโภค, อาญา, สัญญา)
- **📄 เอกสารตัวอย่าง** — ดาวน์โหลดแบบฟอร์มกฎหมาย 6 ประเภทเป็นไฟล์ Word (.doc)
- **🔗 แหล่งอ้างอิง** — รวบรวมเว็บไซต์และหนังสือกฎหมายที่น่าเชื่อถือ
- **👨‍⚖️ หาทนาย** — รายชื่อหน่วยงานให้คำปรึกษากฎหมายฟรี

## 🚀 วิธีใช้งาน

1. ดาวน์โหลดไฟล์ `legal-assistant.html`
2. เปิดด้วยเบราว์เซอร์ (Chrome, Firefox, Safari, Edge)
3. ใช้งานได้ทันที — ไม่ต้องติดตั้ง, ไม่ต้องเซิร์ฟเวอร์

## 🛠️ เทคโนโลยี

| รายการ | รายละเอียด |
|--------|------------|
| Frontend | HTML5 + CSS3 + Vanilla JavaScript |
| AI Engine | Google Gemini 1.5 Flash (ฟรี 1,500 req/วัน) |
| ไฟล์ | ไฟล์เดียว (single HTML file, ~1,800+ บรรทัด) |
| Dependencies | Font Awesome 6, Google Fonts (Sarabun) |

## ⚠️ ข้อจำกัด

ข้อมูลทั้งหมดมีวัตถุประสงค์เพื่อการศึกษาเท่านั้น ไม่ใช่คำปรึกษาทางกฎหมายอย่างเป็นทางการ
หากมีปัญหากฎหมายจริง ควรปรึกษาทนายความที่ได้รับใบอนุญาต
